## ğŸ¨ Customization

### Update Contact Email
Replace `YOUR_EMAIL@example.com` in line 3419 with your actual email address.

### Add Portfolio Images
Replace placeholder images in the carousel section with actual event photos:
```html
<img src="path/to/your/image.jpg" alt="Event Name">
```

### Modify Team Members
Update team member details in the Team Section by editing the data attributes:
- `data-name`
- `data-role`
- `data-department`
- `data-bio`
- `data-image`
